# 散点图 (Scatter Plot)

## 📊 图表简介

散点图是用**坐标平面上的点**来表示两个变量之间关系的图表，每个点的位置由两个变量的数值决定，是探索变量相关性的重要工具。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **相关性分析** | 探索两个连续变量之间的关系（正相关、负相关、无相关） |
| **聚类发现** | 识别数据中的自然分组或簇 |
| **异常检测** | 发现数据中的离群点或异常值 |
| **回归分析** | 配合回归线展示变量关系和预测模型 |
| **分布展示** | 展示大量数据点的整体分布形态 |

## 📈 图表分类

### 1. 基础散点图
- 展示两个变量的关系
- 每个点代表一个观测值

### 2. 分组散点图
- 用不同颜色/形状区分不同组别
- 适合比较不同类别的分布差异

### 3. 气泡图
- 散点图的扩展，用点的大小表示第三个变量
- 同时展示三个维度的信息

### 4. 散点矩阵图
- 多个变量两两组合的散点图矩阵
- 快速探索多变量关系

### 5. 带回归线的散点图
- 添加线性或非线性拟合曲线
- 定量展示变量关系

## ⚖️ 与相似图表的对比

| 对比图表 | 散点图优势 | 散点图劣势 |
|----------|-----------|-----------|
| **折线图** | 不假设数据连续，展示原始分布 | 不适合展示时间趋势 |
| **热力图** | 展示原始数据点分布 | 数据量大时点会重叠 |
| **柱状图** | 展示连续变量关系 | 不适合分类数据比较 |

## 🔄 散点图的组合与增强

### 与密度图组合
- 边缘添加直方图或密度曲线
- 同时展示分布和相关性

### 与回归分析结合
- 添加回归线及置信区间
- 展示预测模型效果

### 与聚类结果结合
- 用颜色标注聚类类别
- 可视化聚类效果

## 🤔 如何选择

```
目的是什么？
├── 探索两变量关系 → 基础散点图
├── 比较不同组别 → 分组散点图
├── 展示三个变量 → 气泡图
├── 探索多变量关系 → 散点矩阵图
└── 建立预测模型 → 带回归线散点图

数据量大小？
├── <500点 → 标准散点图
├── 500-5000点 → 调整透明度
└── >5000点 → 考虑密度图/六边形分箱图
```

## 💡 使用建议

1. **点的透明度**：数据量大时降低透明度，避免遮挡
2. **点的大小**：根据数据量调整，避免过大或过小
3. **添加趋势线**：帮助读者理解整体趋势
4. **标注异常点**：对特殊点进行标注说明
5. **坐标轴比例**：根据数据特征选择合适的比例
6. **分组颜色**：使用对比明显的颜色区分不同组
7. **避免过拟合**：回归线不宜过于复杂，除非有理论支持

## 📐 相关性判断

| 图形形态 | 相关性类型 |
|----------|-----------|
| 从左下到右上的带状分布 | 正相关 |
| 从左上到右下的带状分布 | 负相关 |
| 圆形或无规则分布 | 无相关 |
| 曲线形状 | 非线性相关 |
