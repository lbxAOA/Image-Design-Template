# 等高线图 (Contour Plot)

## 📊 图表简介

等高线图是用**等值线**连接相同数值点，展示**三维数据**在二维平面上的投影的图表，常用于展示地形、温度、压力等连续变化的场数据。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **地形展示** | 地形图、海拔高度分布（经典应用） |
| **气象数据** | 气压、温度、降水量分布 |
| **科学计算** | 函数曲面、优化目标函数 |
| **密度分布** | 二维数据的密度估计 |
| **误差曲面** | 机器学习中的损失函数可视化 |

## 📈 图表分类

### 1. 线条等高线图
- 仅用线条表示等值线
- 简洁清晰

### 2. 填充等高线图
- 相邻等值线之间填充颜色
- 颜色渐变展示数值变化

### 3. 带标签等高线图
- 在等值线上标注数值
- 便于精确读数

### 4. 3D等高线图
- 在三维曲面上绘制等高线
- 同时展示形态和数值

### 5. 密度等高线图
- 用于二维散点的密度估计
- 展示数据聚集区域

## ⚖️ 与相似图表的对比

| 对比图表 | 等高线图优势 | 等高线图劣势 |
|----------|-----------|-----------|
| **热力图** | 展示等值线，便于识别梯度 | 视觉冲击力较弱 |
| **3D曲面图** | 平面展示，无视角问题 | 缺乏立体感 |
| **散点图** | 展示连续场数据 | 不适合展示离散点 |

## 🔄 等高线图 vs 热力图

### 等高线图特点
- 强调**等值线**和**梯度方向**
- 易于识别**极值点**和**鞍点**
- 适合**科学分析**和**精确读数**

### 热力图特点
- 强调**数值大小**的整体分布
- 颜色变化更直观
- 适合**快速概览**

### 组合使用
```
填充等高线图 = 热力图 + 等值线
兼具两者优点
```

## 🤔 如何选择

```
数据类型？
├── 连续场数据（温度、高度）→ 等高线图
├── 离散矩阵数据 → 热力图
├── 需要立体展示 → 3D曲面图
└── 二维散点密度 → 密度等高线图

关注点？
├── 等值线位置 → 线条等高线图
├── 整体分布 → 填充等高线图
├── 精确数值 → 带标签等高线图
└── 极值和梯度 → 等高线图
```

## 💡 使用建议

1. **等高线数量**：根据数据范围选择合适的层级数（10-20条为宜）
2. **等高线间隔**：
   - 等间隔：适合线性变化数据
   - 对数间隔：适合指数变化数据
3. **颜色映射**：
   - 使用感知均匀的颜色方案
   - 低值用冷色，高值用暖色
4. **数值标签**：在关键等高线上添加数值标注
5. **颜色条**：必须添加颜色条说明数值范围
6. **坐标轴比例**：注意X/Y轴比例，必要时设置`axis equal`
7. **网格密度**：影响等高线的平滑程度

## 📐 解读要点

| 图形特征 | 含义 |
|----------|------|
| 等高线密集 | 梯度大，变化剧烈 |
| 等高线稀疏 | 梯度小，变化平缓 |
| 闭合的同心圆 | 局部极值（峰或谷） |
| 鞍形 | 鞍点（一个方向极大，另一方向极小） |
| 等高线方向 | 垂直于等高线方向是最陡方向 |

## 🎨 常用颜色方案

| 颜色方案 | 适用场景 |
|----------|----------|
| `parula` | 通用，默认推荐 |
| `jet` | 科学可视化（经典但有争议） |
| `terrain` | 地形图 |
| `coolwarm` | 正负值分布 |
| `viridis` | 对色盲友好 |

## ⚠️ 注意事项

1. **插值方法**：不同插值方法可能产生不同的等高线
2. **边界处理**：边界处的等高线可能不准确
3. **数据分辨率**：低分辨率数据的等高线可能不平滑
4. **零值处理**：注意是否需要特殊标注零等高线
