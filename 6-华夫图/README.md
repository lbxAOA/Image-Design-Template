# 华夫图 (Waffle Chart)

## 📊 图表简介

华夫图（又称**方块百分比图**）是用**小方格矩阵**来表示比例关系的图表，通常使用10×10=100个小方格代表100%，直观展示各部分的占比。

## 🎯 适用场景

| 场景                 | 说明                                     |
| -------------------- | ---------------------------------------- |
| **百分比展示** | 直观展示各部分占总体的比例               |
| **完成度展示** | 任务/目标的完成进度（如KPI完成率）       |
| **简单构成**   | 少量类别的占比分布（建议≤5类）          |
| **信息图表**   | 在报告或信息图中替代饼图                 |
| **视觉强调**   | 强调某个特定比例（如"每10人中有3人..."） |

## 📈 图表分类

### 1. 标准华夫图（10×10）

- 100个小方格
- 每个方格代表1%

### 2. 自定义网格华夫图

- 根据需求调整网格大小
- 如5×5=25格，每格代表4%

### 3. 图标华夫图（象形图）

- 用图标代替方格
- 更具视觉吸引力（如用人形图标表示人口）

### 4. 多行华夫图

- 多个华夫图并排展示
- 用于对比不同组的占比

## ⚖️ 与相似图表的对比

| 对比图表         | 华夫图优势                     | 华夫图劣势             |
| ---------------- | ------------------------------ | ---------------------- |
| **饼图**   | 更容易准确判断比例，无视觉扭曲 | 不如饼图紧凑           |
| **柱状图** | 视觉效果更有趣，适合信息图     | 精确数值展示不如柱状图 |
| **环形图** | 更直观，无需计算角度           | 占用空间较大           |

## 🔄 华夫图 vs 饼图选择指南

```
什么时候用华夫图？
├── 类别较少（2-5类）
├── 需要精确展示百分比
├── 在信息图/报告中使用
└── 受众对数据可视化经验较少

什么时候用饼图？
├── 强调"部分与整体"关系
├── 空间有限
└── 各部分差异明显
```

## 🤔 如何选择

```
展示目的？
├── 百分比/占比 → 华夫图或饼图
├── 进度/完成度 → 华夫图
├── 精确比较 → 华夫图（优于饼图）
└── 趋势/时间序列 → 选择其他图表

类别数量？
├── 2-3类 → 标准华夫图
├── 4-5类 → 华夫图（需调整颜色对比度）
└── >5类 → 考虑柱状图或其他图表
```

## 💡 使用建议

1. **类别限制**：建议不超过5个类别，否则难以区分
2. **颜色选择**：使用对比明显的颜色，避免相近色
3. **数值标注**：在图例或方格旁标注具体百分比
4. **排列顺序**：通常从左下角开始，从左到右、从下到上填充
5. **方格间隙**：保持适当间隙，增强可读性
6. **图例位置**：放在图表右侧或下方
7. **圆角处理**：轻微的圆角可使图表更美观

## 📐 视觉解读

| 占比 | 华夫图呈现  | 直觉感受 |
| ---- | ----------- | -------- |
| 25%  | 1/4方格填充 | 四分之一 |
| 50%  | 1/2方格填充 | 一半     |
| 75%  | 3/4方格填充 | 大部分   |
| 90%  | 几乎填满    | 绝大多数 |

## 🎯 适合华夫图的典型表述

- "每100人中有XX人..."
- "XX%的用户选择了..."
- "任务已完成XX%"
- "市场份额：A占XX%，B占XX%..."
