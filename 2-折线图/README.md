# 折线图 (Line Chart)

## 📊 图表简介

折线图是用**连续的线段**连接各数据点来展示数据随某一变量（通常是时间）变化趋势的图表，是展示时间序列数据的首选图表。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **趋势分析** | 展示数据随时间的变化趋势（上升、下降、波动） |
| **多系列比较** | 对比多个数据系列的变化趋势 |
| **周期性分析** | 发现数据的周期性规律（如季节性波动） |
| **预测展示** | 展示历史数据与预测数据的对比 |

## 📈 图表分类

### 1. 基础折线图
- 单一数据系列的趋势展示
- 最简洁直观的形式

### 2. 多系列折线图
- 多条折线在同一坐标系中对比
- 适合比较不同组的变化趋势

### 3. 面积图（填充折线图）
- 折线下方填充颜色
- 强调数值的累积效果

### 4. 堆叠面积图
- 多个面积图堆叠
- 展示总量及各部分的变化

### 5. 阶梯折线图
- 用阶梯状线段连接
- 适合展示离散变化的数据（如价格变动）

## ⚖️ 与相似图表的对比

| 对比图表 | 折线图优势 | 折线图劣势 |
|----------|-----------|-----------|
| **柱状图** | 更好展示连续变化趋势 | 不适合展示离散分类数据 |
| **散点图** | 强调数据之间的连续关系 | 不适合探索相关性 |
| **面积图** | 更清晰展示单个系列的变化 | 不如面积图强调累积效果 |

## 🔄 组合与增强

### 1. 折线图 + 柱状图

| 优点 | 说明 |
|------|------|
| 双轴展示 | 柱状图展示绝对值，折线图展示变化率 |
| 量纲分离 | 利用双Y轴展示不同量纲的数据 |
| 一图多用 | 同时呈现"量"和"率" |

**典型应用**：左Y轴（柱状图）展示销售额、产量；右Y轴（折线图）展示增长率、完成率

### 2. 折线图 + 置信区间/误差带

用于展示数据的**不确定性范围**：

| 优点 | 说明 |
|------|------|
| 可信度展示 | 展示预测的置信区间或误差范围 |
| 科学严谨 | 学术论文和预测分析的标准做法 |
| 视觉直观 | 带状阴影一目了然 |

**适用场景**：预测模型结果、实验数据趋势、统计分析展示

### 3. 折线图 + 散点图

用于展示**趋势与原始数据**的关系：

| 优点 | 说明 |
|------|------|
| 拟合展示 | 散点为原始数据，折线为拟合/平滑曲线 |
| 异常识别 | 容易发现偏离趋势的异常点 |
| 数据完整 | 既展示原始数据又展示趋势 |

**适用场景**：回归分析、趋势拟合、数据平滑处理

### 4. 折线图 + 参考线/阈值线

用于**目标对比和阈值标识**：

| 优点 | 说明 |
|------|------|
| 基准明确 | 水平线表示目标值、平均值、阈值 |
| 快速判断 | 一眼看出是否超过/低于标准 |
| 区域划分 | 可配合背景色划分安全/警告/危险区域 |

**适用场景**：指标监控、目标追踪、阈值预警

### 5. 折线图 + 面积填充

用于**强调累积效果或范围**：

| 优点 | 说明 |
|------|------|
| 视觉冲击 | 填充色增强数据的视觉权重 |
| 范围强调 | 两条折线之间填充可强调差异范围 |
| 层次分明 | 堆叠面积展示各部分贡献 |

**适用场景**：累计数据展示、范围比较、构成变化分析

### 6. 折线图 + 标注/注释

用于**标记关键事件或数据点**：

| 优点 | 说明 |
|------|------|
| 事件关联 | 在特定时间点添加事件注释 |
| 峰谷标注 | 标记最大值、最小值、转折点 |
| 解释说明 | 对异常波动添加原因说明 |

**适用场景**：时间序列分析、事件影响分析、异常解释

## 🤔 如何选择

```
数据是否有时间维度？
├── 是 → 数据点密度
│   ├── 稀疏（<50点）→ 带标记点的折线图
│   └── 密集（>50点）→ 纯折线图（无标记）
├── 需要对比几个系列？
│   ├── 1-3个 → 多系列折线图
│   └── >3个 → 考虑分面展示或小多图
└── 需要强调累积？ → 面积图
```

## 💡 使用建议

1. **数据点标记**：数据点少时添加标记，数据点多时可省略
2. **线条样式**：不同系列使用不同颜色或线型（实线、虚线）
3. **Y轴处理**：
   - 一般情况：从0开始
   - 变化幅度小：可截断但需标注
4. **图例位置**：放在不遮挡数据的位置，或直接标注在线条旁
5. **趋势线**：可添加平滑趋势线或移动平均线辅助分析
6. **数据点数量**：避免在一张图中展示过多系列（建议≤5条）
