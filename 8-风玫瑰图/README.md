# 风玫瑰图 (Wind Rose Chart)

## 📊 图表简介

风玫瑰图是一种特殊的**极坐标柱状图**，主要用于展示风向和风速的频率分布，图形形似玫瑰花瓣，故得名。也可用于展示任何具有方向性的周期数据。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **气象分析** | 风向、风速的频率分布（经典应用） |
| **环境评估** | 污染物扩散方向分析 |
| **航空航海** | 机场/港口的风向条件评估 |
| **选址规划** | 风电场、工业厂房的选址依据 |
| **方向性数据** | 任何具有方向属性的数据（如动物迁徙方向） |

## 📈 图表分类

### 1. 基础风玫瑰图
- 仅展示风向频率
- 每个方向用一个扇形表示

### 2. 堆叠风玫瑰图
- 在风向基础上叠加风速分级
- 颜色深浅表示不同风速等级

### 3. 频率风玫瑰图
- 强调各风向出现的频率
- 扇形长度代表频率百分比

### 4. 平均风速风玫瑰图
- 展示各风向的平均风速
- 扇形长度代表平均风速值

### 5. 能量风玫瑰图
- 用于风能评估
- 展示各风向的能量潜力

## ⚖️ 与相似图表的对比

| 对比图表 | 风玫瑰图优势 | 风玫瑰图劣势 |
|----------|-----------|-----------|
| **柱状图** | 方向信息直观，符合实际方位 | 仅适合方向性数据 |
| **雷达图** | 可叠加多个变量（风速等级） | 维度固定（方向） |
| **热力图** | 空间方向感更强 | 不适合展示时间变化 |

## 🔄 风玫瑰图的关键要素

### 方向划分
| 划分数 | 每扇区角度 | 方向精度 |
|--------|-----------|----------|
| 8 | 45° | 低（N, NE, E, SE...） |
| 12 | 30° | 中 |
| 16 | 22.5° | 高（常用） |
| 36 | 10° | 非常高 |

### 风速分级（常用标准）
| 等级 | 风速范围 | 描述 |
|------|----------|------|
| 静风 | 0-0.2 m/s | 烟垂直上升 |
| 软风 | 0.3-1.5 m/s | 感觉有风 |
| 轻风 | 1.6-3.3 m/s | 树叶微动 |
| 微风 | 3.4-5.4 m/s | 树叶摇动 |
| 和风 | 5.5-7.9 m/s | 小树摇动 |
| 清风 | 8.0-10.7 m/s | 大树摇动 |
| 强风 | >10.8 m/s | 行走困难 |

## 🤔 如何选择

```
数据特征？
├── 有方向属性 → 风玫瑰图
├── 无方向属性 → 选择其他图表
│
方向数据类型？
├── 风向+风速 → 堆叠风玫瑰图
├── 仅风向频率 → 基础风玫瑰图
└── 风能评估 → 能量风玫瑰图

时间跨度？
├── 年度汇总 → 单个风玫瑰图
├── 季节对比 → 多个风玫瑰图并列
└── 月度变化 → 12个小图或动态展示
```

## 💡 使用建议

1. **方向零点**：通常北方（0°/360°）置于图顶部
2. **方向顺序**：顺时针排列（符合地理惯例）
3. **颜色方案**：风速从低到高，颜色从冷色到暖色
4. **扇区间隙**：保持小间隙，增强可读性
5. **同心圆参考**：添加频率参考圆环
6. **静风处理**：可在中心区域标注静风频率
7. **方向标签**：标注主要方位（N, E, S, W）

## 📐 数据解读

| 图形特征 | 含义 |
|----------|------|
| 某方向扇形最长 | 该方向是主导风向 |
| 颜色集中在浅色 | 以低风速为主 |
| 颜色深且扇形长 | 强风频繁来自该方向 |
| 图形对称 | 风向分布较均匀 |
| 图形偏向一侧 | 存在明显的主导风向 |
