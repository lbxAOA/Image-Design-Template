# 柱状图 (Bar Chart)

## 📊 图表简介

柱状图是一种用**矩形条的长度或高度**来表示数据大小的图表类型，是数据可视化中最基础、最常用的图表之一。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **分类比较** | 比较不同类别之间的数值差异（如不同产品销量、不同地区人口） |
| **时间序列** | 展示离散时间点的数据变化（如月度销售额、年度增长率） |
| **排名展示** | 直观展示数据的大小排序（如TOP10排行榜） |
| **构成分析** | 堆叠柱状图展示整体与部分的关系 |

## 📈 图表分类

### 1. 基础柱状图
- **垂直柱状图**：最常见形式，适合类别名称简短的情况
- **水平柱状图**：适合类别名称较长或类别较多的情况

### 2. 分组柱状图
- 多个数据系列并排放置
- 适合比较同一类别下不同组的数据

### 3. 堆叠柱状图
- 将多个数据系列堆叠在一起
- 适合展示总量及各部分的构成

### 4. 百分比堆叠柱状图
- 堆叠柱状图的变体，高度统一为100%
- 强调各部分占比而非绝对值

## ⚖️ 与相似图表的对比

| 对比图表 | 柱状图优势 | 柱状图劣势 |
|----------|-----------|-----------|
| **折线图** | 更适合展示离散数据、强调数值大小 | 不适合展示连续变化趋势 |
| **饼图** | 更适合多类别比较、精确数值展示 | 不如饼图直观展示占比 |
| **热力图** | 单维度比较更清晰 | 不适合展示矩阵数据 |

## 🔄 组合与增强

### 1. 柱状图 + 折线图

当需要**同时展示数值和趋势**时，可以组合使用：

| 优点 | 说明 |
|------|------|
| 双轴展示 | 可展示不同量纲的数据（如销量与增长率） |
| 信息分层 | 柱状图展示绝对值，折线图展示变化趋势 |
| 高信息密度 | 一图多用，节省空间 |

**适用场景**：销售额与同比增长率、数量与累计占比

### 2. 柱状图 + 误差线

用于展示数据的**不确定性或波动范围**：

| 优点 | 说明 |
|------|------|
| 精确表达 | 展示标准差、置信区间、最大最小值 |
| 科学性强 | 学术论文和实验报告的标准做法 |

**适用场景**：实验数据展示、统计分析结果、测量误差表示

### 3. 柱状图 + 散点图

用于展示**汇总数据与原始数据**的关系：

| 优点 | 说明 |
|------|------|
| 数据透明 | 在柱状图（均值）上叠加原始数据点 |
| 分布可见 | 可观察数据的离散程度和异常值 |

**适用场景**：样本量较小的实验数据、需要展示个体差异的场景

### 4. 柱状图 + 参考线/阈值线

用于**对比目标或标准**：

| 优点 | 说明 |
|------|------|
| 目标对比 | 快速判断是否达标 |
| 基准明确 | 如平均线、目标线、行业标准 |

**适用场景**：KPI完成情况、及格线对比、行业平均水平比较

### 5. 柱状图 + 数据标签/注释

用于**强调关键数据**：

| 优点 | 说明 |
|------|------|
| 直观展示 | 无需对照Y轴即可读取数值 |
| 重点突出 | 可对特定柱子添加注释说明 |

**适用场景**：报告展示、关键数据强调、异常值说明

## 🤔 如何选择

```
是否比较类别数据？
├── 是 → 类别数量
│   ├── ≤10个 → 垂直柱状图
│   └── >10个 → 水平柱状图或考虑其他图表
├── 需要展示构成？
│   ├── 关注绝对值 → 堆叠柱状图
│   └── 关注占比 → 百分比堆叠柱状图
└── 需要多组比较？ → 分组柱状图
```

## 💡 使用建议

1. **类别数量**：建议不超过10-12个类别
2. **基线从零开始**：避免截断Y轴造成误导
3. **颜色使用**：同组数据使用相同颜色，不同组使用对比色
4. **数据标签**：重要数值可直接标注在柱子上方
5. **排序**：根据需要可按数值大小排序，增强可读性
