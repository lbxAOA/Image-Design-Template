# 蝴蝶图 (Butterfly Chart / Population Pyramid)

## 📊 图表简介

蝴蝶图（又称**人口金字塔**、**背靠背柱状图**）是将两组对比数据以中轴为对称轴，向左右两侧延伸的图表，形似蝴蝶翅膀。

## 🎯 适用场景

| 场景 | 说明 |
|------|------|
| **人口结构** | 男女人口按年龄段分布（经典应用） |
| **双向对比** | 任何需要对称比较两组数据的场景 |
| **收支分析** | 收入与支出的分类对比 |
| **正负值展示** | 同一指标的正面与负面表现 |
| **前后对比** | 同一对象不同时期的数据对比 |

## 📈 图表分类

### 1. 标准蝴蝶图
- 两组数据等比例展示
- 适合数据量级相近的情况

### 2. 百分比蝴蝶图
- 将数据转换为百分比
- 消除量级差异，关注结构对比

### 3. 堆叠蝴蝶图
- 每侧可堆叠多个子类别
- 展示更复杂的数据结构

### 4. 彩色分段蝴蝶图
- 使用渐变色区分不同层级
- 增强视觉效果

## ⚖️ 与相似图表的对比

| 对比图表 | 蝴蝶图优势 | 蝴蝶图劣势 |
|----------|-----------|-----------|
| **分组柱状图** | 对称结构更直观，便于逐项对比 | 只能对比两组数据 |
| **水平柱状图** | 同时展示双向数据 | 不适合单向数据展示 |
| **折线图** | 分类比较更清晰 | 不适合展示趋势变化 |

## 🤔 如何选择

```
是否需要对比两组数据？
├── 是 → 数据性质
│   ├── 对称/对立关系（男/女、收/支）→ 蝴蝶图
│   ├── 多组比较 → 分组柱状图
│   └── 时间序列 → 折线图
└── 否 → 选择单向图表

数据是否有自然顺序？
├── 有（年龄段、等级）→ 按顺序排列
└── 无 → 按数值大小或自定义顺序
```

## 💡 使用建议

1. **保持对称**：两侧使用相同的刻度范围，便于比较
2. **颜色区分**：左右两侧使用对比明显的颜色（如蓝色/红色）
3. **中心轴**：清晰绘制中心分隔线
4. **类别标签**：放在中间，两侧共用
5. **数据标签**：可在柱状图末端标注具体数值
6. **X轴刻度**：显示绝对值而非负数（虽然绘制时使用负数）
7. **图例说明**：明确说明左右两侧代表的含义

## 📐 数据解读

| 图形特征 | 含义解读 |
|----------|----------|
| 底部宽顶部窄 | 年轻群体多，传统金字塔型（发展中特征） |
| 中间宽两端窄 | 中年群体多，纺锤型（老龄化过渡期） |
| 顶部宽底部窄 | 老年群体多，倒金字塔型（深度老龄化） |
| 左右对称 | 两组数据量级接近 |
| 左右不对称 | 两组数据存在显著差异 |
